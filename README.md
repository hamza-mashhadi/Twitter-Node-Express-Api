# Simple Node with Express Server with REST API

## Features

- Express
- REST API

## Requirements

- [node & npm](https://nodejs.org/en/)

## Installation
- `npm install`
- `npm start`
- optional: include _.env_ in your _.gitignore_

### GET Routes

- visit http://localhost:3000
  - /twitter/:queryString
  - /twitter/searchByUser/:userHandle/:queryString
